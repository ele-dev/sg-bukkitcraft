Technik

Der Minecraft Server wird auf einem Fedora 23 V-server von Contabo gehostet.
Mit insgesamt ungefähr 12GB RAM und einer Quadcore CPU läuft unser Server System stabil.
Verwendet wird die kostenlose Spigot Serversoftware auf der aktuellen Minecraft 1.8 Version.

BungeeCord

Um die Last der Spieler und verschiedenen Games möglichst gut zu verteilen, unterteilen wir
den Minecraft Server in mehrere Parts (z.B. Lobby, Minigames, ... ).
Diese Parts werden miteinander verknüpft, so dass die Spieler ohne Probleme vom einen Part in den nächsten geleitet werden können.
Dafür wird die ebenfalls freiverfügbare BungeeCord Software verwendet. BungeeCord läuft genau wie Spigot auf der Java Runtime (JRE).
